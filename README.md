# Password Generator
Простой генератор паролей с графическим интерфейсом на PyQt6. Позволяет создавать пароли разной длины с выбором типа символов.

**Функциональность**
    Программа предоставляет следующие возможности:
    выбор длины пароля с помощью слайдера (от 6 до 32 символов)
    отображение текущей длины пароля на LCD‑дисплее
    три режима генерации паролей через радиокнопки:
        only letters - только буквы (латинские, верхний и нижний регистр)
        only numbers - только цифры
        symbols - буквы и цифры (без специальных символов)
    генерация пароля по нажатию кнопки «Generate»
    отображение сгенерированного пароля в текстовом поле.
**Элементы интерфейса:**
    Слайдер - регулирует длину пароля (6-32 символа).
    LCD‑дисплей - показывает текущее значение длины пароля.
    Радиокнопки - выбор типа символов для генерации:
        only letters
        only numbers
        symbols.
    Кнопка «Generate» - запускает генерацию пароля.
    Текстовое поле - отображает сгенерированный пароль.
